## 二、行為模式：換個名字叫「寄生循環」

中立版叫它「寄宿行為模式」，但我們直說吧——這是一套精準的 **情感寄生術**：

### 寄生循環五步曲

| 步驟 | 行為                         | 直白翻譯                                                  |
| ---- | ---------------------------- | --------------------------------------------------------- |
| 1    | 進入新關係/新家庭            | 物色下一個宿主。標準：有房子住、願意被他利用、最好心軟    |
| 2    | 蜜月期表現良好               | 裝乖。煮飯、拖地、撒嬌、哭泣——全套表演到位                |
| 3    | 開始小謊話、花錢異常         | 原形畢露。因為他從來就不是在「改變」，他只是在「演」      |
| 4    | 已讀不回、態度惡劣           | 面具戴不住了，開始把對他最好的人當出氣筒                  |
| 5    | 被趕出/關係崩壞 → 尋找下一家 | Reset。清除存檔，開始新遊戲。帶走的東西：零成長、新的負債 |

**已完成循環**：
高雄前女友 → 西門前女友 → 阿庭家 → 小恩（中壢/台南）→ 高雄家 → 現任女友

六段關係，六次相同的劇本。如果這是電影早就被觀眾罵爛了——「導演你連劇情都不改一下嗎？」

## 三、「說謊」？不，這是一套作業系統

普通人說謊是偶爾犯錯，戎狄說謊是 **預設模式**。真話才是例外。

### 謊言 vs 現實對照表

| 他說的                             | 真相                       | 毒辣評語                                                   |
| ---------------------------------- | -------------------------- | ---------------------------------------------------------- |
| 「這周才要搬出去」                 | 上個月就沒回家了           | 他連離家出走都要騙人日期——這種謊有什麼意義？純粹是肌肉記憶 |
| 「有存幾萬元」                     | 錢的來源不明，工作頻繁更換 | 存款跟他的承諾一樣——聽聽就好，要求看帳戶就翻臉             |
| 「每個月有還錢」                   | 還款有進帳但不符合收入     | 哇，真棒，還了一點點。剩下的 20 萬什麼時候還？下輩子？     |
| 2025「看起來好轉」                 | 實際完全斷絕家人聯繫       | 「好轉」= 騙術升級了。從演不好的騙子變成演完直接消失的騙子 |
| 博弈欠款自述 14W                   | 老闆說 30W                 | 連自己欠多少錢都說謊，是怕嚇到金主還是自己數學真的不好？   |
| 「只有一個月沒繳房租」被趕出阿庭家 | 可能有其他問題             | 你信嗎？反正小恩信了，然後被坑了兩年                       |

## 五、「迴避型人格」？太客氣了

中立版用了很多學術詞彙：迴避型依附、因應策略、核心信念……

讓我翻譯成人話：

- **「渴望親密但恐懼依賴」** → 想蹭飯但不想洗碗
- **「迴避情感表達」** → 問他愛不愛你，他說「不知道」。翻譯：你是工具人，他懶得演了
- **「在壓力下撤退」** → 一遇到問題就裝死。帳單來了？裝死。要求談話？裝死。契約到期？裝、死
- **「功能性角色取得歸屬」** → 用煮飯拖地當房租——精算下來他連計時工的時薪都不到
- **「失去關係後極度痛苦但不修復」** → 會哭、會喝酒、會消沉——但就是不會道歉、不會改、不會回頭。因為重新找一個比修復舊的容易太多了

## 六、核心信念翻譯

中立版說他可能有以下核心信念：

| 學術說法                 | 毒辣翻譯                                                         |
| ------------------------ | ---------------------------------------------------------------- |

| 「付出不會被感激」       | 所以我乾脆不付出。反正你們欠我的——我從小就沒人愛                 |
| 「我不值得被真正愛」     | 所以好好愛我的人一定有問題，我得趕快破壞掉這段關係來驗證我的假設 |

這三條信念組合起來就是一句話：**「這個世界虧欠我，但我也不敢去要——所以我用偷的。」**

偷感情、偷信任、偷安全感——拿到手之後用到壞掉就丟掉，然後去偷下一個。

## 七、「可憐」不是免死金牌

是，他的童年很慘。母親拋棄、父親嚴厲、17 歲被趕出家門。

但這裡有一個殘忍的事實：**很多人有類似甚至更慘的經歷，但沒有選擇用同樣的方式生活。**

- 他姊姊同樣的家庭出來的，正常過日子
- 小恩也並非一帆風順，但選擇了面對
- 阿庭家的孩子們也各有困難，但沒有人像他一樣

「可憐」是用來理解他為什麼變成這樣，**不是** 用來合理化他繼續這樣。

23 歲了。被趕出家門到現在 6 年。6 年的時間，可以讀完一個學士加碩士。他做了什麼？從一個人的沙發換到另一個人的床。

## 十、最後的最後

中立版說「他確實擁有愛的能力，2024 年初證明了這一點」。

毒辣版的回應是：**那不叫愛，那叫表演慾最旺盛的時期。**

2024 年 1-6 月，他剛來到新環境、有新的觀眾、有新的舞台。他的每一次哭泣、每一次撒嬌、每一次「我好怕你不要我」——是真情實感還是求生本能？

答案可能兩者都是。但問題是：**就算他的感情是真的，保質期也只有半年。**

半年後，面具掉了，謊言來了，迴避開始了。

然後他會找到下一個觀眾，重新開場，繼續演。

## 結語

這不是一個壞人的故事。這是一個 **把所有人都變成配角的人** 的故事。

在戎狄的世界裡，所有人都是佈景——姊姊是童年的佈景、阿庭家是過渡期的佈景、小恩是最華麗的佈景、現任女友是最新的佈景。

他從一個舞台走到另一個舞台，演著相同的戲碼，說著相同的台詞。

唯一改變的是觀眾席的人。

而那些被他用過的佈景，還留在原地，慢慢收拾他離開後的殘局。
