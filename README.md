# 移工經歷網站

## 專案概述

此專案為單頁式靜態網站，主要以時間軸方式呈現長篇敘事內容，並附帶表格化分析區塊與可切換的心理狀態分析報告。

## 檔案結構

```
index.html                                  # 網站主頁（含樣式、內容與互動腳本）
README.md                                   # 專案說明與更新紀錄
人物側寫.md                                  # 人物側寫原始文件
人物側寫1.md                                 # 人物側寫補充文件
20260404與移工現任女友的對話.md                # 對話紀錄
20260405狀態新增.md                           # 狀態更新紀錄
戎狄心理狀態分析_20260405.md                   # 中立醫療版心理分析（完整 Markdown 原始檔）
戎狄心理狀態分析_毒辣版_20260405.md             # 超級毒辣版心理分析（完整 Markdown 原始檔）
戎狄心理狀態分析_終極毒辣真實版_20260405.md      # 終極毒辣真實版心理分析（完整 Markdown 原始檔）
```

## 功能說明

- 時間軸式長篇敘事呈現（一月至尾聲）
- 黏性導覽列與滾動進度條
- 心理狀態深度分析摘要區塊
- **可切換標籤的心理狀態完整分析報告**（三個版本：🔬 中立醫療版、🌶️ 超級毒辣版、💀 終極毒辣真實版）
- RWD 響應式設計（6 組斷點：1200px / 992px / 768px / 600px / 420px / 360px + prefers-reduced-motion）
- 滾動淡入動畫（IntersectionObserver，支援降級與減少動畫偏好）
- 表格列 hover 高亮、標籤切換按鈕微交互動畫
- 標籤按鈕區 sticky 定位，長內容閱讀時可快速切換版本

## 使用方式

直接以瀏覽器開啟 `index.html` 即可檢視頁面。
