## 專案概述

此專案為單頁式靜態網站，主要以時間軸方式呈現長篇敘事內容，並附帶表格化分析區塊與可切換的心理狀態分析報告。部署於 GitHub Pages，使用 `fetch()` 動態載入各區段 HTML 片段。

## 技術架構

### 動態載入機制

`index.html` 為輕量 shell（約 8KB），頁面載入時透過 `fetch()` API 依序載入 `sections/` 內 19 個 HTML 片段，插入對應的 `<div>` 容器中：

1. **Promise 鏈**：確保區段依正確順序載入
2. **`insertAdjacentHTML('beforeend', ...)`**：同一容器可追加多個月份片段
3. **初始化延遲**：滾動進度條與淡入動畫在所有區段載入完成後才初始化

### 注意事項

- **GitHub Pages**：完全相容，同源 `fetch()` 正常運作
- **本地測試**：須透過 HTTP 伺服器開啟（`file://` 協定下 `fetch()` 會被 CORS 阻擋）
  ```bash
  python -m http.server 8080
  ```

## 功能說明

- 時間軸式長篇敘事呈現（一月至 2026/04/08 最新進度）
- 黏性導覽列與滾動進度條
- 心理狀態深度分析摘要區塊
- **可切換標籤的心理狀態完整分析報告**（三個版本：🔬 中立醫療版、🌶️ 超級毒辣版、💀 終極毒辣真實版）
- RWD 響應式設計（6 組斷點：1200px / 992px / 768px / 600px / 420px / 360px + prefers-reduced-motion）
- 滾動淡入動畫（IntersectionObserver，支援降級與減少動畫偏好）
- 表格列 hover 高亮、標籤切換按鈕微交互動畫
- 標籤按鈕區 sticky 定位，長內容閱讀時可快速切換版本

## 使用方式

### GitHub Pages

推送至 GitHub 後，透過 GitHub Pages 直接存取即可。

### 本地開發

```bash
# 啟動本地伺服器
python -m http.server 8080
# 開啟瀏覽器 http://localhost:8080
```
