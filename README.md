# 移工經歷網站

## 專案概述

此專案為單頁式靜態網站，主要以時間軸方式呈現長篇敘事內容，並附帶表格化分析區塊。

## 檔案結構

- `index.html`：網站主頁，包含內容、樣式與前端互動腳本。
- `README.md`：專案說明與更新紀錄。

## 本次整理內容

- 調整整體視覺版型，提升長文閱讀的節奏與留白。
- 改善導覽列黏性效果，並優化手機版橫向捲動操作。
- 縮窄文字閱讀寬度，降低長段落閱讀疲勞。
- 強化時間軸卡片、頁首摘要區與分析表格的層次。
- 修正原有內嵌樣式語法問題。
- 保留原始內容結構，不變動主要敘事文本。

## 使用方式

直接以瀏覽器開啟 `index.html` 即可檢視頁面。
