## 二、行為模式觀察

### 2.1 反覆出現的迴避行為（Avoidance Pattern）

戎狄的行為中最顯著且一致的模式是 **系統性迴避**，表現為：

- **對家人的迴避**：姊姊、伯母的關心訊息長期已讀不回。父親的訊息除法院傳票外一律不回應。2025 年中開始完全不打招呼、不回家。
- **對親密關係的迴避**：當關係進入需要承諾或面對問題的階段，即開始疏遠、態度惡劣、找藉口製造衝突。
- **對責任的迴避**：欠款不面對、工作問題不事先告知、搬家時間說謊、花錢流向不交代。
- **對自我的迴避**：不願回顧過去、不願面對自己的價值觀問題、不願承認需要幫助。

**關鍵特徵**：迴避模式在不同情境（討債壓力、家人關心、親密關係要求）中表現完全一致，顯示這是一種 **結構化的因應策略（coping mechanism）**，而非針對特定事件的反應。

### 2.2 依附模式（Attachment Pattern）

根據觀察，戎狄的依附行為高度符合 **不安全依附——迴避型（Avoidant Attachment）** 的特徵：

| 特徵               | 戎狄的具體表現                                                     |
| ------------------ | ------------------------------------------------------------------ |
| 渴望親密但恐懼依賴 | 交往後會搬進對方家住（寄宿行為），但從不談論未來                   |
| 迴避情感表達       | 說不知道怎麼跟男生/女生表達；被問愛不愛回答「不知道」              |
| 在壓力下撤退       | 面對關心或要求時選擇已讀不回、換帳號、不接電話                     |
| 關係中的功能性角色 | 透過「照顧家務」「顧小孩」取得留在家庭中的正當性，而非透過情感連結 |
| 失去關係後的反應   | 極度痛苦（哭泣、酗酒數月），但不會主動修復                         |

### 2.3 寄宿行為模式（Repeated "Nesting" Behavior）

戎狄有反覆搬入他人家庭並承擔家務角色的行為：

1. **阿庭家** → 幫忙顧小孩、做家事、付生活費 → 被趕出
2. **小恩家（中壢）** → 被照顧、搭伙共居 → 因行為問題搬離
3. **台南住處** → 由小恩安排 → 因說謊和金錢問題搬回高雄
4. **高雄家** → 短暫回歸 → 不打招呼、不回家、再度搬出
5. **現任女友** → 疑似再度同居 → 結果待觀察

**分析**：此行為反映的是 **對「家庭」概念的渴望**，而非對特定對象的情感。戎狄尋找的是「家的功能」（安全、穩定、歸屬），而非「關係中的愛」。一旦環境要求他面對自身問題或承擔情感責任，即觸發迴避機制。

## 三、可能涉及的心理議題

### 3.1 迴避型人格特質（Avoidant Personality Traits）

戎狄的行為模式與迴避型人格障礙（Avoidant Personality Disorder, AvPD）或迴避型依附風格有高度重疊，但需注意：

- 他在特定關係初期（如與小恩 2024 年 1-6 月）**能夠展現情感**——哭泣、撒嬌、表達在乎
- 這顯示其迴避行為可能更偏向 **習得性因應策略**，而非人格結構的核心缺陷
- 在安全且持續的環境中，他曾短暫展現出真實自我

### 3.2 早期依附創傷（Early Attachment Trauma）

- **母親出生後即離開**：最原始的依附對象缺失。雖然姊姊/伯母提供了替代性照顧，但無法完全替代母職的角色。
- **父親的情感隔離**：管教嚴厲但不表達愛，造成「被管控但不被愛」的認知框架。
- **17 歲被趕出家門**：正值青少年自我認同形成期，被驅逐出家門強化了「我不值得被留住」的核心信念。

這些經歷可能形成以下 **核心信念（Core Beliefs）**：

- 「依賴他人最終會被拋棄或出賣」
- 「付出不會被感激」
- 「我不值得被真正愛」

### 3.3 物質依賴的神經學影響

戎狄有以下物質使用史：

| 物質 | 使用情形                                       |
| ---- | ---------------------------------------------- |
| 菸   | 長期重度吸菸，2024 年戒菸後疑似復吸            |
| 酒   | 失戀後長期酗酒、工作壓力下每日飲酒、員購酒事件 |

**神經學觀點**：

- 長期尼古丁與酒精使用會改變大腦的 **獎勵迴路（reward circuit）** 和 **前額葉皮質（prefrontal cortex）** 功能
- 戒斷後的神經傳遞異常可能導致：情緒調節困難、衝動控制下降、延遲滿足能力減低
- 小恩觀察到「近距離相處且持續親密接觸後幾天會比較穩定」——這可能與 **催產素（Oxytocin）** 的分泌有關，親密接觸能暫時穩定情緒調節系統
- 但此效果為暫時性的，無法從根本上修復神經迴路的長期損傷

### 3.4 可能的情緒障礙

根據以下行為跡象，不排除合併情緒相關議題：

- **持續性低自尊**：妄自菲薄（「你不會選擇我」）、認為自己跟小恩很不配
- **陣發性情緒崩潰**：突然大哭 15 分鐘、自述有憂鬱症
- **快感缺失傾向**：周末整天睡覺、沒有自己的生活、不會自主規劃行程
- **人際退縮加劇**：2025 年中起全面斷絕社交與家庭連結

但需注意，上述症狀也可能是 **迴避型因應策略的結果** 而非獨立的情緒障礙。

## 四、行為循環模型

根據所有紀錄，戎狄的行為呈現以下可預測的循環：

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│  1. 進入新環境/新關係                                 │
│     → 表現良好、認真工作、照顧他人                      │
│                                                     │
│  2. 蜜月期（數週~數月）                                │
│     → 穩定、主動、有改變的意願                          │
│                                                     │
│  3. 壓力累積 / 被要求面對問題                           │
│     → 開始出現小謊言、花錢異常、態度轉變                  │
│                                                     │
│  4. 迴避期                                           │
│     → 不接電話、已讀不回、對親近的人態度惡劣              │
│     → 可能復吸（菸/酒）、金錢問題浮現                    │
│                                                     │
│  5. 關係破裂 / 被趕出                                 │
│     → 被動接受、不主動修復、酗酒消沉                     │
│                                                     │
│  6. 尋找下一個「家」                                   │
│     → 新的女友/新的寄宿對象 → 回到步驟 1                │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**已知循環實例**：

- 高雄前女友 → 西門前女友 → 阿庭家 → 小恩（中壢/台南）→ 高雄家 → 現任女友

## 六、預後評估

### 不利因素

- 已形成穩固的迴避循環，且循環週期在縮短（從數月 → 數週）
- 缺乏病識感（不認為自己需要改變或就醫）
- 經濟基礎極度脆弱，無法支撐穩定生活
- 所有願意幫助的人（家人、小恩）均已表示放棄
- 目前新關係的對象無力處理其複雜問題

### 有利因素

- 2024 年 1-6 月曾在安全環境中展現真實情感與改變意願
- 在持續且密集的陪伴下，情緒調節能力可以暫時恢復
- 並非完全缺乏情感能力——他具備愛的能力，但需要極其特殊的條件才能啟動
- 年齡尚輕（23 歲），神經可塑性仍有空間

### 建議方向（如有專業介入機會）

1. **精神科評估**：排除或確認情緒障礙（如持續性憂鬱症、適應障礙）
2. **心理治療**：以依附取向治療（Attachment-Based Therapy）或基模治療（Schema Therapy）處理核心信念
3. **物質依賴評估**：評估菸酒使用對神經功能的長期影響
4. **社工介入**：協助建立穩定的生活結構（就業、財務、居住）

## 七、結論

戎狄的核心議題並非單純的「不負責任」或「懶惰」，而是一套從早期依附創傷中發展出來的 **生存策略**。他用迴避來保護自己不再受傷，用寄宿來填補缺失的家庭感，用謊言來維持表面的和平。這套策略在短期內有效——它讓他在各種困境中存活下來——但長期來看，它摧毀了所有他真正需要的關係。

最令人感慨的是：他確實擁有愛的能力。2024 年初那段日子證明了這一點。但要啟動那份能力，需要極高強度的安全感、持續的陪伴、以及他自身的主動意願——三個條件目前都已不存在。

在沒有專業介入的情況下，根據行為循環模型，預測其當前關係將在 **3-6 個月內** 進入迴避期，並重複過往模式。
